# Element Counter in C

This project was developed as part of my Data Structures (DS) coursework in college.

## Overview

The program accepts an array of integers from the user and counts the number of times a specified element appears in the array. If the element is not present, the program displays an appropriate message.

## Learning Objective

The objective of this practical was to understand array operations and implement a basic searching approach using C.

## Technologies Used

- C Programming Language
- GCC Compiler
